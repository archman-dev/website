# Network and Platform Security

Network and platform security establish the boundaries, controls, and monitoring infrastructure that protect your applications and data. This section covers network segmentation, firewalls, endpoint hardening, containerization security, and the foundational layer upon which application security depends.

**Key Areas**:
- Network segmentation and zero-trust architecture
- Firewalls, Web Application Firewalls (WAF), and API Gateways
- Endpoint hardening and security baselines
- Container and Kubernetes security controls
- Platform and runtime isolation
